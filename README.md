# Measurement Calculator

A responsive, single-page unit converter supporting 7 measurement categories:

- Length
- Weight
- Temperature
- Volume
- Area
- Speed
- Time

## Features

- Convert between 50+ units
- Swap "from" and "to" units with one click
- Real-time conversion as you type
- Mobile-friendly UI

## Usage

Open `measurement-calculator.html` in any browser. Select a category, choose your units, and enter a value.
